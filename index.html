Latitude vs. Weather Elements

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.


After assembling the dataset, we used Matplotlib to plot various weather elements against the latitude of each city. The weather elements we looked at are max temperature, humidity, cloudiness, and wind speed. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations observed.
